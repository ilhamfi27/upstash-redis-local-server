Thank you for contributing to Upstash Redis Local!

## Quick start

```bash
git clone https://github.com/aine1100/Upstash-Redis-Local-server.git
cd upstash-redis-local
go mod download
make build
make test
```

## Running locally

```bash
# Terminal 1 — Redis
docker run -d -p 6379:6379 redis:7-alpine

# Terminal 2 — Server
make run
# or
./bin/upstash-redis-local --redis localhost:6379 --token local-dev-token
```

## Making changes

1. Create a branch: `git checkout -b feature/your-feature`
2. Make changes
3. Run tests: `go test -v ./...`
4. Run vet: `go vet ./...`
5. Push and open a PR

## Code style

- Follow standard Go conventions (`gofmt`, `go vet`)
- Keep functions focused and small
- Handle errors explicitly
- Match existing patterns in `internal/`

## Documentation

Docs live in `docs/` and use [Mintlify](https://www.mintlify.com/docs). When adding features, update the relevant MDX page.

Preview locally:

```bash
cd docs
npm i -g mint
mint dev
# Open http://localhost:3000
```

## Pull request checklist

- [ ] Code compiles: `go build ./...`
- [ ] Tests pass: `go test ./...`
- [ ] Docs updated if behavior changed
- [ ] PR describes the **why**, not just the what
