title: Build from Source
description: Build and run upstash-redis-local from source with Go or Make.
---

## Prerequisites

- Go 1.22+
- Make (optional)
- A running Redis instance on `:6379`

## Build the server

```bash
git clone https://github.com/aine1100/Upstash-Redis-Local-server.git
cd upstash-redis-local
make build
```

Binary output: `./bin/upstash-redis-local`

## Build the CLI

```bash
make build-cli
```

Binary output: `./bin/upstash-local`

## Run locally

```bash
./bin/upstash-redis-local \
  --redis localhost:6379 \
  --addr :8000 \
  --token local-dev-token \
  --readonly-token local-readonly-token
```

## All CLI flags

| Flag | Env var | Default | Description |
| --- | --- | --- | --- |
| `--redis` | `REDIS_ADDR` | `:6379` | Redis server address |
| `--addr` | `UPSTASH_ADDR` | `:8000` | HTTP listen address |
| `--token` | `UPSTASH_TOKEN` | `upstash` | Full-access API token |
| `--readonly-token` | `UPSTASH_READONLY_TOKEN` | — | Read-only API token |
| `--max-retries` | — | `10` | Startup connection retries |
| `--retry-delay` | — | `1000` | Retry delay (ms) |
| `--simulate-quota` | `SIMULATE_QUOTA` | `0` | Daily quota simulation |
| `--simulate-rps` | `SIMULATE_RPS` | `0` | RPS limit simulation |

## Homebrew (macOS)

```bash
brew tap aine1100/Upstash-Redis-Local-server
brew install upstash-redis-local
```

## Run tests

```bash
make test
# or
go test -v ./...
```

## Development workflow

```bash
make run          # build and run server
make build-all    # build server + CLI
make clean        # remove bin/
```
